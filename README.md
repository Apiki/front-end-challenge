# Desafio - Front-end Developer

Tecnologias/libs utilizadas:

- React.js
- uuid (para geração de keys únicas)
- axios
- express-handlebars 
- date-fns (para formatação das datas)
- react-router-dom

### Para instalar as dependências do projeto:

```
npm install
```

### Para rodar a plicação:

```
npm start
```
