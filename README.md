Minha versão do <a href="https://github.com/Apiki/front-end-challenge">Desafio-Front-End-Developer</a>

## Getting Started

```bash
npm install
# ou
yarn install

# para baixar as dependências
```

```bash
npm run build
# ou
yarn build

# para gerar build do projeto
```

```bash
npm start
# ou
yarn start

# Agora é só acessar http://localhost:3000
```
